# MARCS - Web y tienda oficial

- Maquetación: HTML5.
- Estilos: CSS3.
- Interactividad: JavaScript (Vanilla JS).
- Diseño: Responsive.

Este proyecto ha sido desarrollado en colaboración con @Mrcsalf04 para crear su web personal de artista. Incluye una landing page interactiva para lanzamientos y videoclips, junto a una plataforma de venta de merchandaising totalmente para la gestión de productos, todo totalmente responsive.
